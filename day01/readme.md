## 笔记分类说明
- 框架概览：PyTorch 张量概念与基础数据结构

## 关键知识点摘要
- 张量是 PyTorch 的核心数据结构，用于表达多维数组
- torch.tensor 与 torch.Tensor 的创建差异与默认类型
- arange 与 linspace 的线性序列生成方式
- rand、randn 与 randint 的随机张量场景
- zeros、ones、full 与 empty 的常见初始化手段
- type 与各类转换方法的使用要点
